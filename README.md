# ✅ To-Do List Application (C++)

A simple **console-based To-Do List application** built using **C++**.  
This mini project is designed to practice **core C++ programming concepts**, logical thinking, and basic application structure.

---

## 🚀 Features

- ➕ Add new tasks
- 📋 View all tasks
- ❌ Remove completed tasks (if implemented)
- 🧠 Simple and clear program flow
- 💻 Runs in the terminal / command line

---

## 🛠️ Technologies Used

- **C++**
- Standard C++ libraries
- **GCC / G++ compiler**
